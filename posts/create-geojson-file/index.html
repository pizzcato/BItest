制作概览/大屏等页面使用的geojson地图文件

pizzcato May 25, 2021 2021-05-25T10:57:41+08:00
Jul 19, 2021 2021-07-19T15:27:14+08:00 7 min

利用qgis

Tip:
geojson本身数据的空间对象类型都支持，但是目前概览也能够加载的只有点(point） 和 多边形(polygon/multipolygon)
Important:
不论那种方式都建议先简化一下对象再导出

方式一:qgis直连数据库导入

在qgis中设置连接数据库,导入地图空间数据,根据需求添加过滤条件,然后选择图层,导出为geojson文件

方式二:导出边框的wkt数据在qgis中添加图层后导出

简化可以在添加到qgis之前在数据库中处理,也可以导入到qgis以后再处理

  1. 数据库中通过st_astext(pg/gp)或t.geom.get_wkt()(Oracle)等函数/方法获取边框的wkt坐标集
  2. Qgis中通过QuickWKT插件来快速生成边框(注意将需要生成在一个文件中的边框添加到同一个图层文件)

用到的操作

qgis图层添加过滤条件

  1. 右键图层选择Filter...

  2. 在输入框中输入过滤条件

    • 输入的内容为过滤条件,即 where 子句的内容,可以通过双击Fields中的字段名来快速添加入输入框
    • Values可以用来显示Fields的某个示例值(Sample)或全部值(All)

  3. 点击Test可以获取结果的记录数,点击OK来使过滤条件生效

qgis导出geojson文件

  1. 右键图层选择Export->Save Features as...

  2. 设置导出信息

  3. 设计完成后点击OK执行

QuickWKT插件的安装和使用

QuickWKT是一个qgis的第三方插件,用以通过wkt坐标集文本快速创建qgis图层

  1. 安装

    1. qgis菜单栏->Plugins->Manage and install plugins
    2. 搜索QuickWKT选择安装

  2. 使用

    1. 点击WKT按钮

    2. 输入wkt坐标集,输入图层标题,点击ok创建图层

    Important:
    Layer title不同或勾选了Create a new layer for every geometry选项,则导入时会以新的图层而不是添加的现有图层的方式导入

利用qgis简化图层

菜单栏->Vector->Geometry Tools->Simplify…

逻辑说明

  1. 简化数据库的空间对象
  2. 将空间对象转换为geojson元素
  3. 将多个geojson元素拼接在一起,并补充额外需要的信息来,生成页面所需geojson文件的文本

函数说明

  • st_simplify:利用道格拉斯抽稀法简化边框: 对于数据库的区域边框,往往需要满足定位的精度要求从而需要比较高的精度,导致边框的坐标集太大。而页面上由于加载的时延要求和比较粗略的展示需求,不需要特别高的精度。这时候就可以通过算法简化坐标。 9e-4就是简化的容差值,单位同经纬度,可以根据需求来设置,值越大代表处理后的结果越简化
Important:
必须简化坐标,否则过大的geojson文件会导致页面加载缓慢和页面处理压力
  • st_asgeojson:生成空间对象的geojson元素

    生成的结果还需要补充才是完整的geojson文件

  • string_agg:字符串聚合 将多行字符串聚合

进一步简化geojson文件

  • 页面所需的geojson文件是不需要任何换行和缩进的,如果geojson文件存在较多的缩进和换行,可以通过join line1的方式来取消

  • 经纬度的坐标没必要太精细,可以同通过正则来截断坐标,减少文件的大小

    查询替换
    (\d+.\d{6})\d+$1

    其中{6}代表截断到6位小数,足以满足页面的需要
    不同编辑器的正则解析有些区别

  1. vscode可以F1在所有命令中搜索join lines;
    也可以在网上搜索相关的在线工具,如Merge Text Lines ↩
